Git install 
 you can download git for free from the following website: https://www.git-scm.com/


 Using Git with Command line 
 Version check 
        git --version  output: git version 2.30.2.windows.1
 

 Configure Git
 Set username
 git config --global user.name "enter your name"
 Set email
 git config --gloal user.email "enter your email"
 Note: User Global to set the username and e-mail for every repository on your computer

 Set Local user name and email
 Set username
 git config user.name "enter your name"
 Set email
 git config user.email "enter your email"

 if you want remove username and email 
 For global
 git config --global --unset user.email
 git config --global --unset user.name

 for Local
 git config --unset user.name
 git config --unset user.email


 Creating Git Folder
 mkdir your folder name     // mkdir makes a new directory   and cd  changes the current directory
 cd /                     //  for drive change     
 cd your folder name       // for folder change


Initailize Git
git init
